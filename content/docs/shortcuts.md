# Keyboard shortcuts

Press <kbd>?</kbd> in the app to see the shortcuts overlay. Full list:

![Keyboard shortcuts modal — press ? to toggle](/docs/shortcuts.png)

## Tools

| Shortcut | Action |
|----------|--------|
| <kbd>V</kbd> | Pointer tool |
| <kbd>S</kbd> | Add State tool |
| <kbd>T</kbd> | Add Transition tool |
| <kbd>Del</kbd> / <kbd>Backspace</kbd> | Delete selected |
| <kbd>⌘A</kbd> | Select all |
| <kbd>⌘Z</kbd> | Undo |
| <kbd>⌘⇧Z</kbd> | Redo |

## Modes

| Shortcut | Action |
|----------|--------|
| <kbd>1</kbd> | DFA |
| <kbd>2</kbd> | NFA |
| <kbd>3</kbd> | PDA |
| <kbd>4</kbd> | Turing Machine |
| <kbd>5</kbd> | Mealy Machine |
| <kbd>6</kbd> | Moore Machine |
| <kbd>7</kbd> | CFG (Grammar Editor) |
| <kbd>8</kbd> | L-System |

## Panels

| Shortcut | Action |
|----------|--------|
| <kbd>⌘.</kbd> | Toggle Simulation panel |
| <kbd>⌘/</kbd> | Toggle Properties sidebar |
| <kbd>⌘M</kbd> | Toggle Conversions panel |

## Simulation

| Shortcut | Action |
|----------|--------|
| <kbd>⌘ Enter</kbd> | Start simulation |
| <kbd>⌘ '</kbd> | Step simulation |
| <kbd>⇧⌘ Enter</kbd> | Fast run |
| <kbd>⌘0</kbd> | Reset simulation |

## File

| Shortcut | Action |
|----------|--------|
| <kbd>⌘S</kbd> | Share / Copy URL |
| <kbd>⌘E</kbd> | Export JSON |
| <kbd>⌘O</kbd> | Import file |

## Canvas

| Shortcut | Action |
|----------|--------|
| <kbd>Space</kbd> + Drag | Pan |
| Scroll | Zoom |
| <kbd>Shift</kbd> + Click | Multi-select |
| Double-click | Add state / Edit label |
| Right-click | Context menu |
| <kbd>⌘1</kbd> | Zoom to Fit |
| <kbd>⇧⌘L</kbd> | Auto Layout |
| <kbd>⇧⌘X</kbd> | Clear All |
| <kbd>⇧⌘Q</kbd> | Add Trap State |
| <kbd>Esc</kbd> | Deselect / Close |
| <kbd>?</kbd> | Show shortcuts overlay |
